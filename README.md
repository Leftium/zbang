# zBang

zBang is a front-end for Kagi (search + bangs)

## Main features
- Bang filter/search
- Kagi search QoL features

### Bang filter/search
- Instant filtering as you type
- Fuzzy matching (matches highlighted in title, url, trigger)
- Sorted by most popular first
- Find the shortest alternate bangs: duplicate bangs are grouped together

### Kagi search QoL
- Open multiple bangs at once, one tab per bang
- Simple way to type `!` on mobile (double-tap SPACE)
- Multi-line/fullscreen textarea (for AI prompts)
- Handy char/word/line count (for AI prompts)
- Search/browse query history (planned)
