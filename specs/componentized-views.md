# Componentized Views

- A "view" is an interchangable component that renders content based on standardized input passed to it.
- Often a list of items; selecting an item may result in an action

## Examples

### Standardized input
- textarea: textarea value
- nlp: result of passing textarea through compromise nlp (https://github.com/spencermountain/compromise)
- image input?
- clipboard input?
